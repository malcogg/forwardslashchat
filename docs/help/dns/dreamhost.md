# DreamHost — CNAME for `chat.`

1. **Panel** → **Domains** → **Manage Domains** → **DNS** for your domain.  
   Or **Websites** → **Manage** → **DNS** (wording varies).

2. Add a **CNAME**:
   - **Name:** `chat`  
   - **Points to:** `cname.vercel-dns.com`  

3. Save.

## DreamHost nameservers

DNS changes apply when the domain uses **DreamHost’s nameservers**. If you’ve pointed the domain to Cloudflare or another DNS, add **`chat`** there instead.
