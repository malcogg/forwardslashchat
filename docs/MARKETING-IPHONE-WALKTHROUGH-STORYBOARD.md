# iPhone walkthrough — script & storyboard

**Product:** ForwardSlash.Chat — AI chatbot trained on your site, on your domain.  
**Format:** Vertical 9:16 screen recording in an iPhone frame + optional captions.  
**Total runtime (target):** ~90 seconds (tight) or ~2:00 (full). **Audience:** SMB buyer who wants “hands-off” setup.

**Recording tip:** Use Safari on iPhone, signed-in test account, **one continuous take per section** if needed, then edit in CapCut. Use **Screen Recording** from Control Center.

## Production checklist

- [ ] Test account with a **paid or staging** order so dashboard and chat match the story (or label video “demo”).  
- [ ] **Blur or fake** email/phone in checkout if recording public.  
- [ ] **Do Not Disturb** on; brightness ~80%; **Guided Access** optional to avoid accidental exits.  
- [ ] Export **1080×1920** (9:16), add iPhone frame in CapCut (template), **large captions** (safe area).  
- [ ] End card: logo, **forwardslash.chat**, primary CTA.
